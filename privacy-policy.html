6. Other Privacy Policies

Who We Are

Our website address is http://18gogogo.net.

Comments

When users post comments on this website, we collect the data shown in the comment form, the user's source IP address, and the browser's user agent string to help detect spam comments.

This website will create an anonymized string (also known as a hash) based on the user's email address and provide it to the Gravatar service to check if the user is a user of this service. For more information on the Gravatar service's privacy policy, please visit https://18gogogo.net/privacy-policy/. After the website administrator approves the user's comment, the user's profile picture will be publicly displayed in the comment content.

Media

If you upload images to this website, please avoid uploading images with embedded EXIF GPS location data, as other users can download the images and extract the location data from them.

Cookies

When users post comments on this website, they can choose whether to store their name, email address, and website URL in cookies; storing this data is for the user's convenience, so that when posting other comments, they do not need to fill in personal information again. Unless the user manually clears the device's cookies, these cookies will remain on the personal device for one year.

If a user visits the login page of this website, the system will set a temporary cookie to determine whether the user's browser accepts cookies; this cookie does not contain any personal data and will be discarded when the user closes the browser.

After logging in to the website, the system will set several cookies to store the user's login information and display settings; login information cookies will be retained for two days, and display settings cookies will be retained for one year. If the user checks the "Stay logged in" option during login, the user's login status will be maintained for two weeks; after logging out, the login information cookies on the user's device will be removed.

When a user edits or publishes an article, additional cookies will be stored in the browser. This cookie does not contain any personal data and only records the article ID of the article the user is writing, and it will expire after one day.

Embedded Content from Third-Party Websites

Articles on this website may embed videos, images, articles, etc., and the privacy handling of embedded content from third-party websites is exactly the same as when users visit these websites.

Regardless of whether users have accounts on these third-party websites or are logged in, they will collect data related to users in various ways, such as cookies, embedding third-party tracking codes, monitoring user interactions with embedded content, etc.

Who We Share Your Data With

If you request a password reset, your current IP address will be included in the password reset email.

How Long We Retain Your Data

When a user posts a comment on this website, the comment and its metadata will be retained indefinitely. This way, the system can automatically recognize and approve any follow-up comments without holding them in a moderation queue.

For users who register on this website, the website also stores the personal information they provide on the user's [Profile] page. All users can view, edit, or delete their personal information at any time (except for the username, which cannot be changed). Please note that website administrators can also view and edit this personal information.

7. Data We Collect and Privacy Policy

1. Data We Collect

This application collects and processes your data through the following SDKs:

  • 1.1 Google Mobile Ads SDK
    • Data Collected: Anonymous advertising identifier, device information (such as device model, OS version), IP address, advertising interaction data (such as clicks, browsing history)
    • Purpose of Collection: Used to display personalized ads in the application to support the operation and development of the application.
    • Data Processing Method: This data is processed by Google for ad delivery and effectiveness analysis. For details, please refer to the Google Ads Privacy Policy.
  • 1.2 Google Sign-In SDK
    • Data Collected: Google account email address, account ID, access permissions to Google Sheets that you authorize
    • Purpose of Collection: Used for user authentication, allowing you to log in to this application and access data in Google Sheets.
    • Data Processing Method: Email and account ID are used for authentication, and Sheets access permissions are only used to read and write the spreadsheet data you authorize. For details, please refer to the Google API User Data Policy.
  • 1.3 Google Sheets API
    • Data Collected: Google Sheets spreadsheet data that you authorize
    • Purpose of Collection: Used to read and write data from your Google Sheets spreadsheet to implement the core functions of the application (such as querying and updating data).
    • Data Processing Method: Access to spreadsheet data is only after your authorization, not shared with third parties, and only used for display and operation within the application.
  • 1.4 Firebase Authentication SDK
    • Data Collected: Email address, login status, email verification status
    • Purpose of Collection: Used to manage user authentication, ensuring that only verified users can use the application functions.
    • Data Processing Method: Email address is used for registration and login, and Firebase stores your login credentials and performs authentication. For details, please refer to Firebase Privacy and Security.
